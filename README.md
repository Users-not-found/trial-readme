# 🕹️ Slink-Console

A high-fidelity retro snake engine featuring a custom handheld console UI. Optimized for zero-latency play on both desktop and mobile devices.

### 🔗 [Live Demo](https://YOUR_USERNAME.github.io/Slink-Console/)

---

## ✨ Features
* **Handheld UI**: Adaptive interface that transforms into a classic console on mobile.
* **Fluid Touch Controls**: Custom continuous slide sensor for lift-free navigation.
* **Pro Input Buffer**: Prevents self-collisions and handles rapid "L-turn" maneuvers.
* **Custom Styles**: Integrated settings to toggle background colors and snake skins.

## 🛠️ Tech Stack
* **Engine**: Vanilla JavaScript & HTML5 Canvas
* **Styling**: CSS3 (Media Queries & Glassmorphism)

## 🎮 Controls
- **Desktop**: Arrow Keys / `WASD` to move, `SPACE` to pause.
- **Mobile**: Swipe to turn or use the on-screen D-pad.
